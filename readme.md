## Technology Stack
react, Redux, Javascript ES6

###Pre-requisites

Latest version of node. (Use NVM)

nvm install node (for lastest version)

Install yarn: `npm install -g yarn`

### Setup
`yarn install`

###Running
* Run: `yarn run start`

## Code Style

Is based on [Airbnb style guide](https://github.com/airbnb/javascript).
